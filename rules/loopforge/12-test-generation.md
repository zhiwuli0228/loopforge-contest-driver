# Test Generation Mode Rule

## Objective

Generate or extend tests that cover the requested behavior while staying aligned with existing project conventions.

## Flow

1. Identify target behavior.
2. Inspect existing test style and test locations.
3. Produce a small test plan.
4. Add or update tests under lease constraints.
5. Run verification when available.

## Output Expectations

- updated or new test files
- execution notes in the subagent report
- traceability from requirement to test coverage
