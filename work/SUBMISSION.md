# Submission Notes

This package is submitted as a generic Loop Engineering hosting platform.

## Delivered Artifact

- `work/` is the formal submission root.
- `code/` is expected to be provided by the execution platform and is not bundled here.

## Expected Output

After execution, the platform should retrieve:

- the modified `code/` project tree
- `code/.loopforge/reports/final-report.md`

## Non-Goals

LoopForge does not:

- commit code
- push changes
- open pull requests
- upload results to a contest platform
