# Skill Generation Flow

## Applies To

- generating a reusable business skill from a tool or workflow
- codifying repeated engineering procedures into an operational skill
- creating examples and usage guidance for a new specialized capability

## Flow

`CAPABILITY_INVENTORY -> USAGE_CONTRACT -> SKILL_DRAFT -> EXAMPLE_GENERATION -> QUALITY_REVIEW -> FINAL_REPORT`

## Phase Outcomes

- `CAPABILITY_INVENTORY`: identify what the underlying tool can actually do
- `USAGE_CONTRACT`: define inputs, outputs, boundaries, and preconditions
- `SKILL_DRAFT`: write the reusable skill content
- `EXAMPLE_GENERATION`: provide concrete usage examples
- `QUALITY_REVIEW`: check clarity, correctness, and non-overlap with LoopForge core
- `FINAL_REPORT`: summarize delivered skill assets and review status
